# Code of Conduct

## Our Standards

This project maintains a professional environment focused on technical collaboration and constructive feedback.

Expected behavior:

- Professional and respectful communication
- Constructive criticism focused on ideas, not individuals
- Clear and concise technical discussion
- Recognition of different experience levels and perspectives
- Focus on improving the template and helping users

Unacceptable behavior:

- Personal attacks or inflammatory language
- Harassment or discriminatory comments
- Trolling, spam, or off-topic disruption
- Publishing others' private information
- Any conduct inappropriate for a professional setting

## Enforcement

Maintainers have the right and responsibility to remove, edit, or reject comments, commits, code, issues, and other contributions that violate this code of conduct.

Violations can be reported by opening an issue or contacting the maintainers directly. All reports will be reviewed and addressed appropriately.

## Scope

This code of conduct applies to all project spaces including the repository, issue tracker, pull requests, and any other communication channels related to the project.

## Attribution

This code of conduct is focused on maintaining a productive technical environment for building quality software.
